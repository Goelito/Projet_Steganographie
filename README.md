# Description

Cette application permet de cacher un message dans une image ou de lire un message déjà caché à l’aide d’un algorithme de stéganographie.

### Instructions d’utilisation

Renommez le fichier Stéganographie.toto en Stéganographie.exe

### Format des fichiers
Tous les fichiers utilisés doivent être au format PNG. Les autres formats ne sont pas pris en charge.

### Placement des fichiers 
Placez vos images dans le dossier images situé à la racine de l’application.

### Nom des fichiers 
Lorsque vous utilisez l’application, tapez le nom complet du fichier, y compris l’extension .png.
Exemple : monimage.png


Attention : le nom doit être exact pour que l’application puisse accéder à l’image.

### Fonctionnalités

Encoder un message : insérez un texte secret dans une image PNG.

Décoder un message : récupérez le texte caché dans une image PNG.
